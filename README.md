**📂 1. 전체 구조 파악하기 (지형도)**

우리 홈페이지는 여러 개의 파일이 유기적으로 연결되어 있습니다.



index.html: 홈페이지의 얼굴입니다. (공지사항/갤러리 미리보기가 자동으로 뜹니다.)



style.css: 홈페이지의 옷입니다. 색상이나 폰트, 박스 모양을 바꾸고 싶을 때만 건드립니다.



images/: 모든 사진의 저장소입니다. 새로 올릴 사진은 반드시 여기에 넣으세요.



notice\_view/ \& gallery\_view/: 실제 공지글과 갤러리 글이 저장되는 '글 보관함'입니다.



**📢 2. 새로운 공지사항(Notice) 등록법**

우리 홈페이지는 \*\*'동기화 시스템'\*\*을 사용합니다. index.html을 복잡하게 고칠 필요가 없습니다.



파일 복사: notice\_view/ 폴더에서 가장 최근 파일(예: notice\_view\_12.html)을 복사합니다.



이름 변경: 파일 이름을 다음 번호인 notice\_view\_13.html로 바꿉니다.



내용 수정: 메모장이나 코드 에디터로 파일을 열고 <h3> 안의 제목과 <p> 안의 본문 내용을 수정합니다.



메인 연동: index.html 파일 맨 아래쪽 스크립트(script) 부분에서 notice\_view\_12라고 되어 있는 숫자를 13으로만 바꿔주면 메인 화면에 자동으로 뜹니다.



**🖼️ 3. 갤러리(Gallery) 사진 교체법**

갤러리도 공지사항과 원리가 같습니다. 중앙 정렬이 맞춰져 있으니 사진 크기만 신경 써주세요.



사진 준비: 새로 찍은 사진을 images/ 폴더에 넣습니다. (파일명은 영어로! 예: lab\_dinner.jpg)



파일 생성: gallery\_view/ 폴더의 기존 파일을 복사해 새 파일을 만듭니다. (예를 들어 원래 글이 12개 있었으면 새로 추가하는 사진 번호는 13)



경로 수정: 새 파일 안의 <img src="../images/기존사진.jpg"> 부분을 방금 넣은 ../images/lab\_dinner.jpg로 바꿉니다.



메인 연동: 공지사항과 마찬가지로 index.html 하단 스크립트에서 갤러리 파일 번호를 최신으로 업데이트합니다.



**📄 4. 논문(Publications) 업데이트법**

논문은 publications.html 파일을 직접 엽니다.



한 줄 추가: 기존 논문 리스트 한 줄(<li>...</li>)을 통째로 복사해서 맨 위에 붙여넣습니다.



내용 변경: 논문 제목, 저자명, 연도를 바꿉니다.



링크 연결: href="https://doi.org/..." 부분에 해당 논문의 DOI 주소를 넣으면 클릭 시 바로 논문 페이지로 이동합니다.
